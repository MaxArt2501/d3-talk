Transition of what?

  • ‚ö° Numbers: 0 → 100
  • ‚ö° Colors: #f60 → rebeccapurple
  • ‚ö° Lengths: 10px → 120px
  • ‚ö° Strings: M0,20L10,5 → M0,16L12,7

Transition tuning

  • .duration()
  • .delay()
  • .ease() (see d3-ease)

Defining starting values

select('rect')
  .attr('fill', 'red');
transition()
  .select('rect')
  .attr('fill', 'blue');
 
transition()
  .select('rect')
  .attrTween('fill', () =>
    interpolate('red', 'blue')
  );

See also .styleTween() and .tween()
